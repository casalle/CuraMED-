# CuraMED-
Website code for a 'Doctors at door and Telemedicine' service.

I worked mainly on VS code and am new to Github so I am just learning as I go, and will try to make more changes here.
This repository contains the front-end website — a single HTML file that patients and corporate clients can use to learn about our services and get in touch.



**What CuraMED does**

We bring PMDC-certified doctors, nurses, and counselors directly to your home. No waiting rooms, no commute — just professional medical care wherever you are.




**Services offered:**

Doctor Home Visit — PMDC-certified GP at your door in under 90 minutes
Video Consultation — Face-to-face with a doctor via WhatsApp or Zoom
Phone Consultation — Quick medical advice, available 24/7
Psychological Counseling — Confidential sessions with licensed counselors




**Also available for corporate ompanies:**

Corporate & occupational health services
Routine employee screenings and referrals to panel hospitals
Medical fitness certificates and HR health reporting




**What's in this repo**

This is a single-file website — everything lives inside index.html. There are no frameworks, no npm installs, no build steps. Just open the file in a browser and it works.




**Here's what the website includes:**

Hero section — headline, service description, and a Book a Visit button
Booking modal — patients fill in their details and the form sends to WhatsApp and email (via Formspree)
Services section — overview of all four services
How It Works — three-step process explaining how CuraMED operates
Corporate section — information for companies, with a Send Enquiry modal
Pricing section — transparent, upfront pricing for each service
Patient reviews — three testimonials
FAQ — accordion-style answers to common questions
Coverage map — interactive map showing Rawalpindi and Islamabad service areas
Privacy policy and Terms of Service — accessible via footer links
WhatsApp floating button — always-visible quick contact button
Emergency banner — permanent reminder that CuraMED is not an emergency service




**Technical details**

Pure HTML, CSS, and JavaScript — no frameworks or dependencies
Google Fonts (DM Sans + DM Serif Display)
Formspree for form-to-email submissions
WhatsApp click-to-chat API for booking notifications
OpenStreetMap embedded for the coverage area map
SEO meta tags, Open Graph tags, and Twitter Card tags included
Google Analytics placeholder included (replace GA_MEASUREMENT_ID)
Fully responsive — works on mobile, tablet, and desktop



**How to use**

To view locally:
Just download index.html and open it in any browser. No server needed.

To go live:
Upload index.html and your images/ folder to any static hosting service
Recommended free options: Netlify, Vercel, or GitHub Pages
Point your domain (curamed.pk) to your hosting provider
